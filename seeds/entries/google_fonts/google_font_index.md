# 欧文フォント一覧の取得

## アンケートに使用出来る欧文フォント一覧を取得します。

`GET https://creativesurvey.com/api/v1/google_fonts`

~~~

リクエスト例
$ curl -X GET https://creativesurvey.com/api/v1/google_fonts \
-u "auth_token=YourAuthToken":

レスポンス例
[
  {
    "id": 95,
    "name": "Vollkorn",
    "image": "Vollkorn.png"
  },
  {...},
  {...}
]
~~~
